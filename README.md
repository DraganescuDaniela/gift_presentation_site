# A Gift to Share 🕊️

> *"A gift to share is like a smile to give"*

A static gift inspiration website in Romanian that helps you find the perfect gift for anyone. Browse curated ideas by category, get tips on how to choose, or let the **Ruleta Alegerilor** (Choice Roulette) decide for you.

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Home — guide on how to choose the perfect gift |
| `Biblii.html` | Books — Bibles, Christian fiction, children's books, bookmarks, and more |
| `her.html` | For Women — kitchen, gardening, sweets, handmade, clothing, accessories, aesthetic packages |
| `him.html` | For Men — tools, games, grooming products, handmade, aesthetic packages |
| `copii.html` | For Children — toys, sweets, handmade items, aesthetic packages |
| `Felicitari.html` | Greeting cards |
| `ruletaF.html` / `ruletaB.html` / `ruletaC.html` | Choice Roulette — spin to get a gift category suggestion |
| `login.html` / `signup.html` | Login and signup pages |

## Tech Stack

- Plain HTML5 & CSS3 — no frameworks, no build tools
- Font Awesome 4.7 for icons
- Organized CSS in the `css/` folder (navbar, gallery, buttons, layout)

## Project Structure

```
├── index.html          # Home page
├── css/                # Stylesheets
├── acasa/              # Home page images
├── bibli/              # Book images (bibles, novels, children's, etc.)
├── copii/              # Children's gift images
├── f/                  # Flower/art images
├── handmade/           # Handmade gift examples
├── her/                # Women's gift images
├── him/                # Men's gift images
├── Ruleta/             # Roulette assets
└── poze/               # Miscellaneous photos
```

## Running Locally

No build step needed — just open `index.html` in a browser, or serve with any static file server:

```bash
npx serve .
```
